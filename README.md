# Number Properties Analyzer

## Description
A C++20 console application that analyzes an integer and determines a variety
of mathematical classifications and representations of the number.

The program evaluates whether the number belongs to several well-known
number categories and computes additional numerical properties such as
binary representation, digit counts, and parity.

## Features

### Number Classification
The program determines whether the input number is:

- Odd or even
- A triangular number
- Prime or composite
- A square number
- A power of two
- A factorial value
- A Fibonacci number
- Perfect, deficient, or abundant

### Additional Numerical Properties
The program also computes and displays:

- Even parity bit based on the binary digit sum
- Number of decimal digits
- Whether the number is palindromic
- Full 64-bit binary representation
- Decimal representation with thousands separators

## Purpose
This project was developed as a programming exercise to practice numerical
algorithms, number theory concepts, and console-based program design in C++.
